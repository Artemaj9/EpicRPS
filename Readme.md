# EpicRPS
Mobile game "Rock Paper Scissors". Application for Apple devices on the iOS platform.
Play alone or with a friend. Track the statistics of your victories.
🪨 📄 ✂️

## Technologies
- Swift, SwiftUI
- Combine
- iOS 15.0 +
